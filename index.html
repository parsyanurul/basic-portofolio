Tentang Saya

Halo! Nama saya Parsya Nurul Azka, seorang siswi SMK jurusan Pengembangan Perangkat Lunak dan Gim. Saya memiliki passion dalam pengembangan perangkat lunak dan desain gim. Selamat datang di portofolio saya!

Proyek Saya

MyFitness - Aplikasi Pelacak Kesehatan

Deskripsi: MyFitness adalah aplikasi mobile yang saya kembangkan untuk membantu pengguna melacak aktivitas fisik, asupan makanan, dan kemajuan kesehatan mereka. Aplikasi ini menawarkan fitur seperti pelacakan kalori, rencana latihan, dan grafik kemajuan. Dikembangkan menggunakan Flutter dan Firebase.

Fitur Utama: Pelacakan aktivitas, Rencana latihan, Notifikasi pengingat, Integrasi dengan perangkat wearable.

QuestWorld - Gim 2D Petualangan

Deskripsi: QuestWorld adalah gim 2D petualangan yang saya buat menggunakan Unity. Gim ini menawarkan pengalaman petualangan yang seru dengan berbagai level, musuh, dan tantangan. Fitur utama termasuk sistem pertarungan, pengumpulan item, dan berbagai level dengan desain grafis yang menarik.

Fitur Utama: Sistem pertarungan berbasis aksi, Level yang dapat dieksplorasi, Sistem pengumpulan item, Desain grafis pixel art.

Web Portfolio - Situs Web Pribadi

Deskripsi: Situs web ini adalah portofolio pribadi saya yang dirancang untuk menampilkan proyek-proyek yang telah saya kerjakan. Dibangun dengan HTML, CSS, dan JavaScript, situs ini menyediakan informasi tentang saya, proyek-proyek saya, dan cara untuk menghubungi saya.

Fitur Utama: Desain responsif, Galeri proyek, Informasi kontak, Navigasi intuitif.
